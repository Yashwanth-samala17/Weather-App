# 🌦️ Weather-App 
A sleek, real-time weather forecasting application built with web technologies.

## 🚀 Live Demo
[View Live Project](https://yashwanth-samala17.github.io/Weather-App/) 

## ✨ Features
- **Real-time Data:** Fetches live weather updates using the OpenWeatherMap API.
- **Dynamic UI:** Backgrounds/icons change based on weather conditions.
- **Location Search:** Get weather info for any city globally.
